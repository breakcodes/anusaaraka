'trunk' शब्द का सूत्र बनाने का प्रयास:-

उदाहरण:

"1.तना"
The trunk of that tree is strong.
उस पेंड़ का तना मजबूत हैं.

"2.सूंड"
Elephant's trunk is large.
हाथीं की सूंड़ लम्बीं हैं.

"3.बड़ा बक्सा"
keep the extra utensils in the trunk.
बचे र्बतनो को बड़े बक्से में रख दो.

"4.धड़-{बिना हाथ~पांव}"
Human body part

"5.डिक्की"
keep that luggage in the car trunk.
उस सामान को कार की डिक्की में रख दो.


यहाँ "trunk" शब्द विशलेषण करने पर ऐसा लगता है कि इसका विकास 'truncate' -- 'काँट~छाँट{ऊपर-नीचे~से}~करना'.
ऐसा मानने से उसका सूत्र कुछ इस प्रकार से सामने आता हैं    

        truncate
           |
           |      
     |-------------------|
     |                   |
धड़{बिना हाथ~पाँव}       तना     
     |                   |
     |                   |
बड़ा~ बक्सा=ट्रंक       सूंड़
     |                   |
     |                   | 
   डिक्की            जाँघिया


इन उधाहरणो को दृष्टि में रखकर इसका सूत्र "ट्रंक" माना गया हैं.


सूत्र : ट्रंक`
