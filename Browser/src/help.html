Why multiple Rows?

Any machine translation system consists of several modules. However these modules are not robust. They are prone to errors. When a input in typical machine translation system passes through all these modules, the errors get cascaded, and output thus produced is many times meaningless.

Anusaaraka interface presents the output at all these intermediate layers in a readble format in various rows. The user has complete control over the display of rows.

What does each row specify?

  • Row 1: Original English sentence (A)
  • Row 2: Cautionary Signs (B)
    • Shows the cautionary sign for a word indicating special attention.
  • Row 3: Hindi padasutra (C)
    • Least fragile layer.
    • contains Hindi padasutra (word formula) for each English words.
      E.g. small -> chotaa^alpa
      rats -> cuuhaa{s}
  • Row 4: Root (D)
    • Shows root corresponding to the English word.
  • Row 5: Dictionary (E)
    • Shows all the meanings.
  • Row 6: POS tag (F)
    • Shows the best probable Part Of Speech in a given context.
  • Row 7: Chunker Marking (G)
    • Shows the names of phrases (noun phrase, verb phrase, etc).
  • Row 8: Word Grouping (H)
    • The group of words which as a group give some new meaning(e.g. compounding) are grouped together.
      For example, in the above sentence,
      are + ing = 0_rahaa_hai
  • Row 9: Sense Disambiguation (I)
    • Attempts to select the appropriate sense according to the context.
      For example, bank of the river -> kinaaraa nadii kaa
  • Row 10: Preposition Movement (J)
    • The prepositions are moved to their correct Hindi positions.
      onthebank Original English
      ->para--kinaaraaAfter Word Sense Disambiguation
      --+2---kinaare_paraAfter Prep Movement
  • Row 11: Hindi Generation (K)
    • Correct forms of Hindi words are generated using the gender number person agreement information.
      E.g. 'cuuhaa{s}' is changed to 'cuuhe'.
  • Row 12: Suggestions (L)

How to control number of rows?

Use the Show/Hide button in the bottom frame to control the display of rows.

Top

What do the colors specify?

The colors in the output indicate category of the word(s). Following color scheme has been used.

noun-phrase
verb-phrase
adjectival-phrase
adverbial-phrase
preposition
conjunctions
interjections

The categories are marked by the POS taggers and the chunkers. One can see the actual POS tags and the chunker markings by unhiding the corresponding rows in the output.

Top

What do the numbers in the output stand for?

The numbers in lightgreen strip indicate the Paragraph, Sentence, and Row numbers. For example 10.5.G stands for the Gth row of the 5th sentence in 10th paragraph. One can use paragraph and sentence number information to jump to a particular location using GOTO button in the top frame. Thus if you want to jump to say 5th sentence in the 10th paragraph, just type in 10.5 and click on GOTO button.

How do I submit suggestions for Hindi meanings?

Your suggestions for improving the anusaaraka output quality are most welcome.

To record your suggestion, open the suggestion row. Go to the word for which you want to give your suggestion. Type in your suggestions in the suggestion row and to submit, press enter.
